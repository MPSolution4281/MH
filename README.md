# MH AutoTeknik og Optimering

Statisk multi-page hjemmeside til MH AutoTeknik og Optimering med fokus på chiptuning, motoroptimering, værkstedsservices og dansk SEO-indhold.

## Projektstatus

Projektet bruges i øjeblikket som en statisk HTML/CSS/JS-løsning, der kan åbnes med Live Server.

Der ligger også rester af en tidligere Next.js-struktur i repoet (`app/`, `components/`, `data/`, `lib/`, `public/`), men den aktive version af hjemmesiden er den statiske løsning i projektroden.

## Aktive filer

- `index.html`
- `chiptuning.html`
- `find-din-bil.html`
- `services.html`
- `om-os.html`
- `kontakt.html`
- `blog/`
- `styles.css`
- `script.js`

## Kørsel lokalt

Den letteste måde at se siden lokalt på er med VS Code Live Server eller en anden simpel statisk server.

Eksempel:

1. Åbn projektmappen i VS Code
2. Højreklik på `index.html`
3. Vælg `Open with Live Server`

## Struktur

- `images/` indeholder lokale billeder og logo
- `blog/` indeholder blogoversigt og artikler
- `styles.css` samler styling
- `script.js` håndterer delt header/footer, hero-slider, blog-spotlight og bilfinder

## Indhold og rettigheder

Projektet indeholder virksomhedsbranding, originale marketingtekster og indhold tilpasset MH AutoTeknik og Optimering. Brug og viderefordeling bør kun ske med tilladelse fra ejeren.

## Noter

- Kontaktoplysninger og lokation er stadig under løbende afklaring
- Den aktive løsning er lavet til hurtig publicering som statisk site
- Repoet kan senere ryddes op, hvis den gamle Next.js-struktur ikke længere skal bevares
